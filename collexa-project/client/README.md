# Collexa Frontend

React frontend for Collexa - VIT Student Marketplace

## Setup

1. Install dependencies:
```bash
npm install
```

2. Update `.env` if backend is on different URL

3. Start development server:
```bash
npm run dev
```

4. Open http://localhost:5173

## Features

- Browse listings with filters
- User authentication  
- Create/manage listings
- View seller contact info
- Responsive design

## Tech Stack

- React 18
- React Router v6
- Tailwind CSS
- Axios
- Vite
